# Data Directory

Place camera datasets here.

Expected structure:

- `data/camera_1/train/images`, `data/camera_1/train/labels`
- `data/camera_1/valid/images`, `data/camera_1/valid/labels`
- `data/camera_1/test/images`, `data/camera_1/test/labels`
- `data/camera_2/train/images`, `data/camera_2/train/labels`
- `data/camera_2/valid/images`, `data/camera_2/valid/labels`
- `data/camera_2/test/images`, `data/camera_2/test/labels`

You can download reference datasets using `scripts/yolo_datasets_download.sh`.
